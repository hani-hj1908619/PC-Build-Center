This mini intel build is a cute little m-ATX case packed with just enough power for daily tasks using Intel's Core i5-8400 2.8GHz processor combined with 8GB 2666MHz RAM. Thanks to the GTX 1050Ti, it can also handle some light gaming. Although water cooling is uneccessary for a build like this, it's a neat design addition. Using an SSD over an HDD gives a huge boost in overall performance as well.
